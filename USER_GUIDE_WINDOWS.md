# MedPrepAI Windows User Guide (Non‑Tech)

## Quick Start
1. Download `MedPrepAI.zip` and unzip it.
2. Open the folder and double‑click `MedPrepAI.exe`.
3. When Windows asks about camera/microphone access, click **Allow**.

## Using the App
1. Go to **Knowledge Base** tab.
2. Click **Browse** and select your **Questions PDF**.
3. Click **Browse** and select your **Answers PDF**.
4. Click **Parse + Auto Map**.
5. Click **Save Knowledge Base**.
6. Go to **Interview** tab and click **Start Interview**.

## Microphone + Camera
- If the mic button says dependencies missing, the app build is incomplete. Ask for a new build.
- If the camera is black, close other apps using the camera and restart MedPrepAI.

## Troubleshooting
- “Windows protected your PC”: click **More info** → **Run anyway**.
- No camera: Windows Settings → Privacy & Security → Camera → Allow app.
- No microphone: Windows Settings → Privacy & Security → Microphone → Allow app.
